STAT COMMAND

Lo STAT command permette di controllare velocemente il valore della statistica indicata per una certa entità, e il bonus ad essa correlata. Il bonus correlato ad una statistica è dato dal suo valore meno 10.

Il campo [STAT] deve essere l'abbreviazione di una statistica.
Guardare la pagina Home per vedere la lista delle statistiche e le loro abbreviazioni.

La sintassi è la seguente:

>>>   [ NOME ]   [ STAT ]

NOTE: Il campo [STAT] è case-insensitive.

ESEMPIO:

Per controllare il valore di Intelligenza di Sirio:

>>> Sirio int

Output:

 >>> Sirio INT
 
 Sirio INT ==> 13
 Sirio INT bonus:  +3
 

CHECK STAT COMMAND

Il CHECK STAT command permette di fare eseguire una prova ad un'entità, sulla base della statistica indicata.
Il programma simula il tiro di un dado a 20 facce (1d20) e somma il bonus della statistica rilevante. Il risultato della prova viene comparato con il valore della CD (Classe Difficoltà) indicata: se maggiore, l'entità ha superato la prova, altrimenti ha fallito.
La sintassi è la seguente:

>>>   [ NOME ] check [ STAT ] CD [ CD value ]

NOTE: Il campo [STAT] è case-insensitive. Anche la keyword "cd" può essere utilizzata al posto di "CD".

ESEMPI:

Per fare eseguire una prova di Forza CD 15 a Lily:

>>> Lily check FOR CD 15

Output:

 >>> Lily check FOR CD 15
 
 Rolling 1d20... -->  18
 Adding Lily FOR bonus: -2
 Lily FOR check is  16
 Lily has SUCCEDED!
 

VS COMMAND

Il VS command permette di eseguire una prova contrapposta tra due entità, sulla base della statistica indicata.
Entrambe le entità eseguono una prova sulla base della statistica rilevante.
L'entità che ottiene un risultato maggiore vince. In caso di parità, nessuna delle due entità vince, e si ha un pareggio.
La sintassi è la seguente:

>>>   [ NOME ] vs [ NOME ] check [ STAT ]

NOTE: Il campo [STAT] è case-insensitive.

ESEMPI:

Per fare eseguire una prova contrapposta di Carisma, tra Sirio e Lily:

>>> Sirio vs Lily check car

Output:

 >>> Sirio vs Lily check car
 
 Sirio rolls 1d20 --> 10 -3 ==> 7
 Lily rolls 1d20 --> 5 +3 ==> 8
 Lily WINS OVER Sirio
